# bun-range-macro

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

This project was created using `bun init` in bun v1.2.18. [Bun](https://bun.sh)
is a fast all-in-one JavaScript runtime.
